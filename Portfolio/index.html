    flashCode

    flashCode was created to help individuals study common programming terms using virtual flashcards as an alternative to traditional pen and paper methods.

    When visiting flashCode users intially have the option to register with an email or log in as a guest. Once logged in, the user has access to six decks of cards, each representing a different programming language and vocabulary. When a user selects a card deck, they are redirected to that specific deck. The card face shown presents the term and when the user hovers over the card, the corresponding definition and example is revealed. The user will also be presented with book resource options related to the programming language as well as previous, next, and review buttons. The previous and next buttons help the user access the next or previous cards in the deck. When clicked, the review button saves the specified card to the review tab located on the navigation bar. The User can access their saved terms on the review page where they can choose to delete specified terms when no longer needed.

    Skill Exchange

    Skill Exchange is a website that provides users with a non monetary based economy where they can trade skills/lessons with other users.

    When users land on our website they have the option to register or log in. If this is your first time visiting Skill Exchange you'll need to register with a valid email address (so other users can contact you). Once registered, the user is able to login where they are then redirected to the home page where they can add a skill to teach. They can select skills from a variety of categories and add a description about their specific skill and duration of time which they are available to teach it. After entering their first skill the user can either enter another skill or go to the User Profile page to see all of the skills they are currently offering for trade. On the user profile page they can update or delete their submissions. When users are ready to make a trade with another user, they should navigate to the Trade page. The Trade page allows users to view and select from a variety of categories in which they would like to learn a new skill. Once a category is selected, multiple listings of lessons offered by other users in that category are displayed along with an email button. A user can then select the email button which will populate a prefilled email to the user offering that specific skill in order to request an exchange!

    H-Town Hubbub

    H-Town Hubbub is an application created during the 2019 Houston Hackathon that is designed to help the city address the issue of noise pollution.

    In an effort to combat and visualize noise pollution in Houston, I created a React/Node application that uses your device's geolocation data and microphone to record your location along with ambient decibel rating. This information is then displayed using Uber's React Map framework to show a heat map distribution of decibel intensity. Constant exposure to volume over 85 decibels can cause permanent damage to our ears! By crowd sourcing data from Houstonians we can create a map that helps the city know where they should focus their efforts to lower noise pollution.

    Wayfinder

    Wayfinder is an application designed to help companies track and monitor the health of their clients web applications using Google's Lighthouse Audit Tool.

    Designed as a tool to help Poetic Systems monitor the health of their clients web applications, Wayfinder combines the power of React/Redux with the speed and efficency of GraphQL. When a website is added via Wayfinder an audit is run using Google Chrome's Lighthouse. This audit returns important information about the websites health including: SEO, Performance, Accessibility and Best Practices for desktop and mobile. Once a website is added there are two options for retriving updated performance statistics. You can either refresh all the reports via the refresh button or allow the cron job to manually update the reports every seven days. Please refer to the GitHub for login credentials.

  • June 2019

    Fishing App

    Fishing App is a Redux/React application designed to assist fisherman track their catches using geolocation data and keep up to date with current weather conditions.

    Fishing App utilizes the users geolocation data to accurately capture the latitude and longitude along with important attributes about their catch (Type of Fish, Size, Type of Bait, etc...). This data is stored in a database then rendered using Uber's React Map in order to give a visual representation of where the user has been catching fish. Fishing App also calls a weather API in order to give the user a three day forecast of weather conditions so they can better plan their outing. Fishing App also features authenticated routes so users will be able to save their fishing data relative to their individual accounts. This project could eventually serve as a basis for other types of sport hunting and feature multiple types of game on the map overlay.
